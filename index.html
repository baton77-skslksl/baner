HTML/CSS код баннера

Скопируйте этот код, чтобы добавить баннер на свой сайт:

<!-- Баннер с гиперссылкой --> <a href="https://ваш-сайт.com" class="banner" target="_blank"> <div class="banner-content" style=" background: linear-gradient(135deg, #6a11cb 0%, #2575fc 100%); padding: 30px; text-align: center; color: white; border-radius: 10px;"> <div style="font-size: 1.8rem; font-weight: bold; margin-bottom: 10px;"> Заголовок вашего баннера </div> <div style="font-size: 1rem; margin-bottom: 15px;"> Описание баннера с призывом к действию </div> <div style=" display: inline-block; background-color: white; color: #333; padding: 10px 25px; border-radius: 30px; font-weight: bold;"> Нажмите здесь </div> </div> </a> <style> /* Стили для баннера */ .banner { display: block; text-decoration: none; transition: transform 0.3s, box-shadow 0.3s; } .banner:hover { transform: translateY(-5px); box-shadow: 0 10px 20px rgba(0, 0, 0, 0.1); } </style>

Как это работает:

  • Тег <a> создает гиперссылку (атрибут href указывает URL назначения)
  • Атрибут target="_blank" открывает ссылку в новой вкладке
  • Внутри ссылки размещается содержимое баннера
  • CSS стили определяют внешний вид и анимации
  • Измените цвета, текст и URL под свои нужды
